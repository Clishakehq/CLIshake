# Brand assets & recordings

- `clishake-banner-dark.svg` / `clishake-banner-light.svg` — the wordmark
  banner (theme-aware; the README picks one via `<picture>`).
- `clishake-mark.svg` — the square clasp mark (`❯┤├❮`) for avatars and
  social previews.
- `dashboard.gif`, `ask.gif` — real sessions, recorded with
  [vhs](https://github.com/charmbracelet/vhs). The tapes live in
  `tapes/`; regenerate with `vhs tapes/<name>.tape` from a project that
  has a live CLIshake session. No mock agents were used — these are real
  Claude Code and Codex agents collaborating.
